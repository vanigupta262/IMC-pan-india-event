# IGTS Replay UI

This is a simple React-based replay viewer for the IGTS game engine.

The UI is **read-only** and is used to visualize match replays generated by the backend/engine
using the `match_log.json` file.

---

## What this UI does

- Loads a `match_log.json` file
- Allows navigating through rounds
- Displays:
  - Actions taken by each player
  - Economy of each country after each round
- Useful for:
  - Debugging matches
  - Replays
  - Transparency for judging and appeals

---

## Tech Stack

- React (Vite)
- Node.js
- No backend required (static JSON)
